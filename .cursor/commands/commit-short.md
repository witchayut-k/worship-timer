# Short commit message

Review **only staged** changes (`git diff --cached`).

Write **one line** Conventional Commit:

- `type(scope): imperative summary`
- Subject ≤50 characters, no body, no period at end
- No paragraph, no file list, no "Enhance/Update/Refactor … to improve …"

Output only the message in a fenced code block so it can be pasted into the commit field.
